# gost

gost = go + gist

GitHubのgistにコードアップロードするcli

## Installation

```
$ go get github.com/konojunya/gost
```

## Usage

```
$ gost /file/to/path [options]
```

### Options

input description

```
$ gost /file/to/path -m "description message"
```

private gist

```
$ gost /file/to/path -private
```

## Development

```
$ git clone https://github.com/konojunya/gost.git
$ cd gost
$ dep ensure
```
